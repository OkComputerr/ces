# 查询指标列表<a name="ZH-CN_TOPIC_0020098248"></a>

## 功能介绍<a name="section66578044"></a>

查询系统当前可监控指标列表，可以指定指标命名空间、指标名称、维度、排序方式，起始记录和最大记录条数过滤查询结果。

## URI<a name="section62331491"></a>
